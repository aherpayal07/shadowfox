# Contact Management System

A simple Java command-line Contact Management System developed as part of the **ShadowFox Java Development Internship**.

## Features

- Add a new contact
- View all contacts
- Update an existing contact
- Delete a contact
- Store contacts using ArrayList
- User-friendly menu-driven interface

---

## 📌 Technologies Used

- Java 17
- Maven
- ArrayList
- Object-Oriented Programming (OOP)
- Visual Studio Code

## ▶️ How to Run

### 1. Clone the repository

```bash
git clone https://github.com/aherpayal07/shadowfox.git
```

### 2. Open the project

Open the project in Visual Studio Code or any Java IDE.

### 3. Compile the project

```bash
mvn clean compile
```

### 4. Run the application

```bash
mvn exec:java
